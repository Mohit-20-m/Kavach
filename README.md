# 🛡️ KAVACH — Supply Chain Security

Agentic Behavioral Shield for Open Source Supply Chain Security.

KAVACH intercepts every `npm install` and `pip install` and scans it with 5 parallel AI agents before any code executes.

## Install

**macOS / Linux:**
```bash
curl -sSL https://raw.githubusercontent.com/Mohit-20-m/Kavach/main/installer/install.sh | bash
```

**Windows (PowerShell as Administrator):**
```powershell
iex (irm https://raw.githubusercontent.com/Mohit-20-m/Kavach/main/installer/install.ps1)
```

## VS Code Extension

Search **KAVACH** in VS Code Extensions marketplace.

## Disable / Enable

```bash
kavach-disable    # Turn off interception
kavach-enable     # Turn back on
```
